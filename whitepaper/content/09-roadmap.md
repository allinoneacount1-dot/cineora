# 8. Roadmap

The roadmap below uses **quarters and years, not "soon"**. Each phase names concrete deliverables and the metric by which its completion is judged.

## Phase 1 — Genesis (Q3–Q4 2026)

**Objective.** Ship a working on-chain civilization with one canonical world.

**Deliverables.**

1. Anchor program set deployed to mainnet: `cineora-core`, `cineora-living-scene`, `cineora-royalty`, `cineora-legacy`.
2. Living Scene NFT spec v1 with mutable on-chain state and PDA-derived commitments.
3. Aurora Engine v1 (R3F + Three.js + WebGPU fallback) rendering at 60 fps on reference hardware.
4. Three-agent mesh (Director, Screenwriter, Cinematographer) running in simulation-first mode.
5. Closed Creator Council of 50 founding creators onboarding the first 50 worlds.
6. Three independent audits completed (Trail of Bits, OtterSec, Neodyme); bug bounty live on Cantina.
7. $CINE public sale and TGE.

**Exit criterion.** 50 active worlds, 10,000 verified wallet holders, zero SEV-1 incidents over a 90-day observation window.

## Phase 2 — Aurora (2027)

**Objective.** Full agent mesh and the Memory Chain.

**Deliverables.**

1. All sixteen agents operational with bounded autonomy budgets.
2. Memory Chain v1 with content-addressed append-only logs and Merkle-rooted on-chain commitments.
3. World Builder public release, allowing creators to instantiate worlds without code.
4. Reputation Token contract and decay schedule live.
5. World Licensing Network v1 — inter-world treaty protocol.
6. First cross-protocol integration with a tier-1 generative model provider.

**Exit criterion.** 5,000 active worlds, 250,000 monthly active audience members, 100+ licensed derivative works.

## Phase 3 — Cinema (2028)

**Objective.** Distribution at population scale.

**Deliverables.**

1. Virtual Cinema rooms supporting both synchronous and asynchronous viewing.
2. Cinematic Commerce integration with two major payment rails.
3. Mobile-first R3F viewer (iOS, Android) using WebGPU where available.
4. Civilization Analytics dashboard with privacy-preserving aggregate metrics.
5. First academic partnership for a peer-reviewed study of protocol-mediated cultural memory.

**Exit criterion.** 1,000,000 monthly audience members, 50,000 active creators, USD 25M in annual protocol revenue.

## Phase 4 — Civilization (2029+)

**Objective.** Cross-reality reach and story inheritance.

**Deliverables.**

1. Cross-Reality Portal reference implementation (browser ↔ headset ↔ spatial display).
2. Story Inheritance System v1 — automatic dormancy detection, custodian rotation, canonical handover.
3. Civilization Pool fully funded and governed by Evolution proposals.
4. First Legacy Constitutions tested across a 36-month creator absence.

**Exit criterion.** At least ten civilizations operating continuously for 36+ months under their original Legacy Constitution. That number — ten — is small on purpose. Permanence is not a function of volume.
